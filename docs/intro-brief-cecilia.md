# DoodleIP — 项目简介

## 一句话

**不露脸，也能让别人记住你。** 给内容创作者的专属 IP 生成器。

## 产品是什么

DoodleIP 帮内容创作者用 AI 生成一个专属的涂鸦角色（IP），然后把这个角色融入到小红书轮播图里。用户只需要输入文字内容，就能得到一组带有自己 IP 角色的、可以直接发布的轮播图。

## 目标用户

小红书上的内容创作者，两类人：

1. **已经在发内容的** — 干货/观点/教程博主，但全是纯文字截图，在信息流里没有辨识度
2. **想开始但没迈出第一步的** — 有内容能力，但不想露脸、不会设计，不知道怎么建立视觉风格

共同点：有内容，缺视觉身份。

## 解决什么痛点

纯文字内容在小红书信息流里是隐形的。用户刷过就忘了你是谁。

像「机器坏人」那种博主 — 一个简笔 IP 角色贯穿所有帖子 — 让人一眼认出「又是这个人」，辨识度和粘性完全不同。但普通人做不出来：不会画、不会设计、没有工具能从零拥有一个 IP。

DoodleIP 就是这个工具。

## 竞争对手

不是 Canva，不是任何排版工具。我们的竞争对手是**「继续发没有辨识度的纯文字」这个现状**。

## 支持的 IP 风格方向

简笔画、像素风、扁平插画、水彩手绘、潮玩 3D、漫画线稿 — 用户可以选择自己喜欢的风格。
